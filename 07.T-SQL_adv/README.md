## DB Join 정리
#### 💊 Join이란?<br/>
둘 이상의 테이블을 연결해서 데이터를 검색하는 방법<br/>
연결을 하기 위해서는 테이블들이 적어도 하나의 컬럼을 공유해야 하고 이 컬럼을 PK 또는 FK값으로 설정<br/>
